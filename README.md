## Installation and Setup Instructions

Clone down this repository. You will need get composer installed globally on your machine.

Installation:

`composer install`

To Run Test Suite:

`php artisan test`

To Start Server:

`php artisan serve `

To Visit App:

`http://127.0.0.1:8000/superhero/`
